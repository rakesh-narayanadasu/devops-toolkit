JDK (Java Development Kit): is used to **build** Java applications.

Includes:
- JRE
- Java compiler (javac)
- Development tools (debugger, jar tools, etc.)

JRE (Java Runtime Environment): is used to **run** Java applications, not build them.

Includes:
- JVM (Java Virtual Machine)
- Core libraries
- No compiler
- No development tools

In one sentence

JDK = build + run Java apps.

JRE = only run Java apps.
